Description
-----------
This is a sample WordPress plugin that demonstrates the usage of custom functionality.

Installation
------------
1. Upload the `coupon-group` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

Frequently Asked Questions
--------------------------
1. How do I create a new Coupon Group ?
   - Go to the 'Coupon Group' menu in your WordPress admin, and you'll find the New Group' page.
